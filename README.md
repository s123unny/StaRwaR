# StaRwaR
## 2018 CSIE CAMP interactive-course

### Installation
首先在電腦上安裝Node.js

  $ git clone https://github.com/s123unny/StaRwaR.git

  $ cd StaRwaR

  $ npm install

### Usage

  $ node server.js

  打開瀏覽器，連到localhost:9488

  使用routes/controller.js的密碼和對應的id組合登入

  小隊名稱可以自取
